About This Project

I built my personal portfolio site over the course of a 12 week coding bootcamp. I tried out a few different designs & built several mockups in Figma and Adobe XD before landing on the simple, one page site you see today.

My biggest goal was to build a responsive site completely from scratch that would really test the raw skills I learned during bootcamp. i.e. I used CSS Grid and media queries with a dash of vanilla Javascript versus relying heavily on libraries. Not that libraries are bad, of course- but to make sure I had a thorough understanding of the basics!
